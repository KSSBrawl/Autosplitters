# Autosplitters

## VVVVVV
I made an autosplitter for VVVVVV, and might make more for other games in the future.  
Features:
- Starts on new game
- Splits on crewmate rescue
- Splits on trinkets, and a few other useful events
- Resets on exiting to menu
- Uses in game timer
